## Privacy

Animus has no telemetry and no cloud layer of its own. Your code reaches only the AI provider whose API key you configure. For a full breakdown — including local-model operation and how Animus differs from cloud AI tools that collect code for training — see **[Privacy & Data Policy](../guides/privacy.md)**.
